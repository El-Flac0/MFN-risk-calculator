Results are grouped and summed by generic name, separately for Part B and Part D. First search loads each dataset once (~5–15 s), then stays cached.

Part B covers drugs administered in physician/outpatient settings. Part D covers retail/mail-order prescriptions (Overall aggregate row only).
Spending values are summed across all brand variants sharing the same generic name, within each part separately.
△ Projected values use ordinary least-squares linear regression on 2019–2023 actuals, floored at $0. 🌐 GLOBE = Part B projected 2026 spending ≥ $100M. 🛡 GUARD = Part D projected 2026 spending ≥ $69M. Source: data.cms.gov.
